💰 ExpenseWise – Expense Tracking Web Application

ExpenseWise is a full-stack expense tracking web application that helps users manage their personal finances in a simple and organized way. It allows users to track income and expenses, set budgets, and visualize financial data using interactive charts.

🚀 Features
🔐 User Authentication (Signup / Login with JWT)
💵 Add Income & Expenses
📊 Interactive Dashboard with financial summary
📂 Transaction Management (view, filter, search)
🎯 Budget Management with category limits
📈 Analytics & Charts (monthly trends, category breakdown)
🔎 Filters & Search for better tracking
📱 Responsive UI (works on mobile & desktop)
🛠️ Tech Stack
Frontend
React.js
TypeScript
Tailwind CSS
Recharts (for charts)
Backend
Node.js
Express.js
JWT Authentication
Database
MongoDB (with Mongoose)
📊 How It Works
User signs up or logs in
JWT token is generated and stored
User adds income/expenses
Data is stored in MongoDB
Dashboard and analytics update automatically
